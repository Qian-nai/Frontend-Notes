CSS盒模型示例

Box 1 (box-sizing: border-box)

设置width: 300px，实际占用宽度就是300px（包含padding和border）

内容区域。设置了border-box后，width/height包含padding和border。

Box 2 (默认content-box)

设置width: 300px，实际占用宽度是350px（额外加上padding和border）

内容区域。默认情况下，width/height只包含内容，padding和border会额外增加尺寸。

实际应用：卡片组件

卡片标题

卡片内容。使用padding控制内边距，margin控制外边距，border添加边框。
